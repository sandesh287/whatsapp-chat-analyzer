# 📊 WhatsApp Chat Analyzer

A Streamlit-based web application to analyze WhatsApp chat exports and generate meaningful insights.

## 🚀 Features

* 📌 Message statistics (messages, words, media, links)
* 📊 Most active users
* ☁️ Word cloud generation
* 😂 Emoji analysis
* 📅 Monthly & daily timelines
* 📈 Activity heatmaps
* 📆 Weekly & monthly activity patterns

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Matplotlib / Seaborn

## ▶️ How to Run

```bash
git clone https://github.com/your-username/whatsapp-chat-analyzer.git
cd whatsapp-chat-analyzer
pip install -r requirements.txt
streamlit run app.py
```

## 📂 Input Format

* Upload exported WhatsApp chat `.txt` file
